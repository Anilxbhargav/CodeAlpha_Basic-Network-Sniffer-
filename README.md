# Basic Network Sniffer

## Description
This project captures and analyzes network packets using Python and Scapy.

## Features
- Captures live packets
- Displays source IP
- Displays destination IP
- Displays protocol information

## Technologies Used
- Python
- Scapy

## Installation

```bash
pip install scapy
```

## Run

```bash
python sniffer.py
```

## Output
Displays packet information in terminal.
